# Lotus Routes

A beautiful web application showcasing authentic South Asian heritage experiences through music, dance, cuisine, and traditional arts.

## About

Lotus Routes connects cultural enthusiasts, tourists, students, retirees, and families with master artists, renowned chefs, and celebrated performers for immersive South Asian cultural experiences.

## Features

- 🏠 **Home Page** - Introduction to Lotus Routes with featured experiences
- 🎭 **Experiences Page** - Browse and filter cultural experiences by category
- 📝 **Request Form** - Book your cultural experience
- ✅ **Confirmation Page** - Booking confirmation and details
- 📖 **About Page** - Learn about our mission and story

## Tech Stack

- React 18
- TypeScript
- Tailwind CSS v4
- React Router v7
- Vite
- Lucide Icons

## Getting Started

### Prerequisites

- Node.js 18+ or pnpm

### Installation

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build for production
pnpm build

# Preview production build
pnpm preview
```

## Deployment

This project is optimized for deployment on Vercel.
